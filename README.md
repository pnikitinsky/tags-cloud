# TagsCloud

Pavel Ninitinskiy's implementation of frontend

## Run

Install dependencies
```shell
npm install
```

Development mode with hot reload
```shell
npm start
```
## Custom Theming

Add a parameter to the query to enable a different theme
```shell
?theme=custom 
or 
http://localhost:3000/tags?theme=custom
```

## Test

Execute a command
```shell
npm run test
```
